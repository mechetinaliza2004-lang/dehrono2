

## Структура

```
dehrono/
├── index.html            ← только разметка (HTML)
├── css/
│   └── style.css         ← все стили
├── js/
│   └── script.js         ← весь JavaScript
├── assets/
│   ├── fonts/            ← сюда кладутся шрифты JustBreathe и Greybeard
│   ├── images/           ← все картинки сайта отдельными файлами
│   └── video/            ← сюда кладётся видео 2004.mp4
├── .nojekyll             ← нужен для GitHub Pages
├── .gitignore
└── README.md
```

HTML, CSS и JavaScript разнесены по отдельным файлам — так принято
оформлять веб-проекты, чтобы было удобно редактировать стили или
скрипты, не копаясь в разметке.

Все изображения вынесены в `assets/images/` отдельными файлами с понятными
именами (`charm-1-pink-fly.png`, `poster-1.jpg`, `childhood-bag.jpg` и т.д.) —
так удобнее их заменять и поддерживать репозиторий в порядке.

## Иконки папок

На рабочем столе две папки («Подробная информация» и «Детство») используют
картинку `assets/images/folder-icon.png`. Положи туда свою картинку с этим
именем — она подхватится автоматически на обеих папках. Подробности — в
`assets/images/folder-icon-README.txt`.

## Видео

Окно «2004.mp4» теперь настоящий видеоплеер со звуком, громкостью и
перемоткой. Положи видеофайл в `assets/video/2004.mp4` — он подхватится
автоматически, без правок кода. Подробности — в `assets/video/README.txt`.
Пока файла нет, в окне показывается картинка-заглушка.

## Шрифты

- **Just Breathe** — слово «ДЕХРОНО» (логотип).
- **Greybeard 14px** — пункты меню: О НАС, ШАРМЫ, УКРАШЕНИЯ.

Файлы шрифтов кладутся в `assets/fonts/` (см. инструкцию в `assets/fonts/README.txt`).

## Запуск

Открыть `index.html` в браузере — этого достаточно.

## Публикация на GitHub Pages

1. Залить файлы в репозиторий.
2. Settings → Pages → Source: ветка `main`, папка `/ (root)` → Save.
3. Через минуту сайт будет доступен по адресу
   `https://<твой-логин>.github.io/<название-репозитория>/`.
