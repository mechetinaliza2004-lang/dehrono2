Чтобы заменить иконку папки на свою картинку — положи файл
сюда с ИМЕНЕМ ТОЧНО ТАКИМ:

  folder-icon.png

(можно и .jpg, но тогда в коде в index.html нужно поменять
расширение в двух местах: ico" src="assets/images/folder-icon.png")

Эта картинка используется СРАЗУ ДВА РАЗА на сайте:
1. Папка «Подробная информация» (рядом с О НАС)
2. Папка «Детство» (на рабочем столе)

Рекомендуемый размер: квадратная картинка минимум 200×200px
(на сайте она показывается уменьшенной, примерно 94px шириной).
Фон лучше прозрачный (PNG), чтобы папка не была в рамке.

Если хочешь РАЗНЫЕ картинки для каждой из двух папок —
скажи мне, и я заведу два отдельных файла и пути в коде.
